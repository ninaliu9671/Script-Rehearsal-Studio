# 戏剧排练室 · 多幕兼容版

一个**零依赖、单 HTML 文件**的戏剧台词排练工具。上传任意剧本，选择角色，浏览器内即可开始对词练习——其他角色由 TTS 语音自动朗读，轮到你的台词时暂停等待。

> **[在线体验](https://ninaliu9671.github.io/Script-Rehearsal-Studio/)** &nbsp;|&nbsp; 或下载 `戏剧排练室_多幕兼容版.html`，双击用浏览器打开即可离线使用。推荐 Chrome / Edge。

---

## 功能一览

### 剧本上传与解析
- 支持 `.txt` / `.md` / `.docx` 格式
- **单文件**：用 `### 第一幕 ###` 等标题分幕，自动拆分
- **多文件**：每个文件视为一幕，按文件名数字排序
- 内置 AI 提示词模板，可一键复制发给 AI 将原始剧本转换为标准格式

### 角色与语音
- 自动识别全剧所有角色，左侧角色表展示完整演员表
- 每个角色可独立选择 TTS 声线，附带试听功能
- 一键检测浏览器可用语音，筛选真正能发声的自然语音

### 台词练习
- **全显示 / 全隐藏**：一键显示所有你的台词（方便浏览），一键重新遮住（开始练习）
- **单行控制**：每句隐藏台词右侧有「💡 提示」按钮，点击显示；再点「🙈 隐藏」可重新遮住
- **独立「✅ 我说完了」按钮**：说完台词后点击继续，不与提示功能混淆
- **快速跳转**：上一句 / 下一句我的台词
- **搜索**：输入关键词搜索角色名或台词内容，高亮匹配结果

### 舞台提示
- 支持 `〔...〕`、`（...）`、`(...)` 三种括号格式
- 保留显示，斜体灰色样式区分
- 可一键隐藏 / 显示所有舞台提示

### 多幕支持
- 自动识别多幕结构，下拉框切换幕次
- 切幕时角色下拉自动更新为当前幕的角色

### 自动保存
- 剧本内容、角色选择、音色配置、排练进度、舞台提示开关状态——全部自动保存到浏览器 localStorage
- 刷新页面或下次打开，自动恢复上次的练习状态

## 快速开始

1. 下载 `戏剧排练室_多幕兼容版.html`
2. 用浏览器打开（推荐 Chrome / Edge）
3. 上传你的剧本文件（或点击「📖 加载示例剧本」体验）
4. 在左侧角色表中为每个角色选择声线
5. 在「我的角色」下拉菜单中选定你要扮演的角色
6. 点击「▶ 开始排练」

## 剧本格式要求

```
### 第一幕 ###
〔黎明，城堡外。〕
角色A：这是角色A的台词。
角色B：这是角色B的台词。（看向远方）
〔角色A走近〕
角色A：这是另一句台词。

### 第二幕 ###
角色B：第二幕开始了。
```

- 每句台词一行，格式为 `角色名：台词内容`（中英文冒号均可）
- 舞台提示用 `〔〕`、`（）` 或 `()` 包裹，单独一行
- 多幕用 `### 第X幕 ###` 分隔

如果你的原始剧本格式不规范，可以使用页面内置的 **AI 提示词模板**，将剧本发给 AI 自动转换。

## 技术细节

- **纯前端**：单个 HTML 文件，无需安装、无需服务器
- **TTS**：使用浏览器原生 [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/SpeechSynthesis)
- **外部依赖**：仅 [mammoth.js](https://github.com/mwilliamson/mammoth.js)（CDN 加载，用于解析 .docx 文件；不上传 .docx 则无需联网）
- **数据存储**：所有数据保存在浏览器 localStorage，不上传任何内容到服务器
